# 2026-06-02 早间 brief

**地缘政治与国际关系**

美伊局势再起波折。六月一日至二日，伊朗击落一架美军 MQ-1"捕食者"无人机，美军随即对伊朗境内的雷达与无人机控制设施实施轰炸予以回应。尽管出现这一轮军事交火，特朗普仍表示美伊谈判正以"很快的节奏"推进，而伊朗方面则放话不排除在"其他战线"还手。换句话说，谈判桌与战场这两条线眼下同时在动，停火前景依旧不稳。

俄乌战事也未停歇。六月一日至二日，俄罗斯对乌克兰发动大规模袭击，至少造成十一人死亡，前线的消耗仍在持续。

拉美方面，哥伦比亚总统选举出现新变化：以强硬打击犯罪为旗号的政治素人阿韦拉尔多·德拉埃斯普里利亚（Abelardo de la Espriella）在六月一日的选情中领跑。

**经济与市场**

油价延续跌势。六月一日（周一），西得克萨斯中质原油下跌约百分之一点七三，收于每桶八十七点三六美元；布伦特原油下跌约百分之一点七七，收于九十二点〇五美元。即便当天伊朗与美军发生无人机交火，市场仍在继续排除最坏的供应中断情景，押注美伊正走向某种协议。

**科技与 AI**

英伟达在 COMPUTEX 上兑现了它的发布。六月一日，黄仁勋的主题演讲落地了几项重头：面向智能体计算的新一代平台 Vera Rubin 已进入全面量产；消费端推出名为 RTX Spark 的笔记本与台式机，主打在本地运行智能体 AI、不必再为云端算力账单买单，直接对标苹果、英特尔与高通；另有面向 Windows 开发者的 DGX Station，以及用于机器人、强调对物理世界进行仿真理解的 Cosmos 3。整场演讲的主线是：智能体 AI 已经能用、而且能挣钱，"每一个 token 都是一个营收单位"。

**科学与健康**

胰腺癌治疗传来一则少见的好消息。六月二日，一种名为 daraxonrasib 的实验性药物公布数据，在晚期胰腺癌患者身上把生存期延长了一倍，研究者认为它对肺癌、结肠癌与卵巢癌可能同样有效。胰腺癌历来预后极差、进展凶险，这类把生存期翻倍的结果在该领域并不多见，值得继续追踪。

**社会与政治**

两场选举值得记一笔。六月二日，美国加利福尼亚州举行初选，焦点之一是谁将在现任州长纽森任满之后接棒。与此同时，埃塞俄比亚也在进行大选，登记选民超过五千万，是非洲一场规模可观的民主实践。
