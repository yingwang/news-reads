# 2026-06-06 早间 brief

**地缘政治与国际关系**

中东的停火在六月五日继续以纸面的形式存在，战场却又一次走向反面。当天，以色列对黎巴嫩南部的空袭造成至少二十一人死亡，遇难者中包括西顿（Sidon）市政委员会的一名成员、两名叙利亚儿童和一名急救人员；港城提尔（Tyre）另有七人在夜间空袭中丧生。以军同日警告利塔尼河以北三个村庄的居民撤离，预示新一轮针对真主党的行动。换句话说，六月四日刚被黎巴嫩总统称作"最后一次机会"的停火安排，到这一天几乎已被现实抛在身后。

伊朗一侧的对峙也在升级。美军方面通报，在击落德黑兰射向霍尔木兹海峡方向的数架无人机之后，美军打击了伊朗境内的海岸监视雷达站点。与此同时，伊朗外长阿拉格齐（Abbas Araghchi）称谈判"没有任何实质进展"，并警告说，以色列若把对真主党的攻势延伸到贝鲁特，将触发美伊冲突的"全面重启"。需要记住的背景是，美伊名义上的停火自四月八日维持至今，但双方零星交火近日明显转密。

俄乌战线则是僵持与零星打击并存。六月五日清晨，俄军无人机击中基辅州布罗瓦里（Brovary）一处食品加工厂，造成四人死亡、七人受伤；苏梅州在过去二十四小时另有两名平民死亡、十四人受伤。乌方称当夜拦截了俄军发射的一百九十八架无人机。战场评估机构 ISW 判断，俄军推进已大体陷入停滞，原因是乌克兰持续切断其向前线输送补给与兵力的能力。值得另记一笔的是，特朗普在同日表态支持俄乌直接谈判，称局势"已接近解决"。

**经济与市场**

这一天真正的主角，是美股的一场重挫。六月五日，纳斯达克综合指数下跌百分之四点一八，收于两万五千七百零九点四三，创下自二〇二五年四月以来的最差单日表现；标准普尔五百指数下跌百分之二点六四，收于七千三百八十三点七四，中止了此前连续九周的周线涨势；道琼斯工业平均指数下跌六百九十五点一五，即百分之一点三五，收于五万零八百六十六点七八。费城半导体指数录得自二〇二〇年三月以来的最大单日跌幅，一天之内抹去约一点三万亿美元的芯片股市值。

点燃这场抛售的有两根导火索。其一在芯片：博通于六月四日盘后给出的第三财季人工智能芯片收入指引为一百六十亿美元，明显低于市场预期的一百七十二亿，这一"差一口气"的数字把前一日开始的芯片股疲软放大成全板块的踩踏，英伟达当日一度下跌约百分之三点七。其二在宏观：美国五月非农就业新增十七万两千人，是市场预期八万五千人的两倍有余，失业率回落到百分之三点四。强劲的就业把本就渺茫的降息预期彻底浇灭，反而把加息重新摆上桌面，利率期货一度计入高达百分之九十八的加息概率，美国国债收益率随之跳升。下一道关口仍是六月十六至十七日的美联储议息会议，新主席凯文·沃什的取态会被这份就业数据进一步检验。

**科技与 AI**

这场抛售的另一层意味，落在人工智能这条主线上。过去一年多里，市场为人工智能基础设施的巨额投入给出了极高的估值，而六月五日的下跌，第一次以这样的规模把"这笔钱要花多久、回报几何"的疑问摆到台前。归根结底，引爆点正是博通那份偏弱的人工智能芯片指引：当一家核心供应方对下一季度的人工智能订单给出低于预期的数字，投资者会顺势追问，整条算力扩张的曲线是不是被高估了。与此同时，强劲就业推高的加息预期抬高了贴现率，对依赖长期叙事的高估值科技股形成双重挤压。换句话说，这一天压垮芯片股的，并不是人工智能需求本身转弱，而是市场对"增长速度"与"资金成本"两端同时重新定价。

**科学与气候**

医学领域有一则值得记下的进展。研究人员首次在人体上测试了一款由人工智能设计的通用冠状病毒疫苗，初步结果显示其安全且耐受良好。与只针对单一毒株的传统路线不同，这种"通用"设计意在覆盖一整个病毒家族；倘若后续试验顺利，应对未来变种的方式有可能因此被改写。

天文方面，围绕美国国家航空航天局罗曼太空望远镜（Roman Space Telescope）的一项分析称，它有望发现约十万颗系外行星，数量超过此前所有探测任务的总和。这一估算之所以引人注意，是因为它把系外行星的样本量从"成千"一举推向"十万"的量级，为统计意义上的行星普查打开了空间。

此外，此前逼近日本南部的台风"蔷薇"（Jangmi）近日继续北上，给南部地区带来强风与暴雨。
