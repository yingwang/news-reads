# 2026-06-09 早间 brief

**地缘政治与国际关系**

中东的局势在六月八日到九日之间又掉了一次头，这一回是朝缓和的方向。前两日伊朗与以色列刚刚打破四月停火、互射导弹，到了这两天，双方却又不约而同地踩了刹车。以色列总理内塔尼亚胡表示，以方已停止对伊朗的攻击；伊朗也暂停了对以色列的行动，但同时撂下话，倘若以色列在黎巴嫩南部的打击继续，它就重新开打。换句话说，火暂时停了，引信却还连在黎巴嫩那条线上。

美国总统特朗普把这次转向往前推。他称以伊双方正"寻求立即停火"，关于"和平"的最后谈判正在进行，并扬言美国不久将在伊朗问题上宣布"全面胜利"。不过他也表明，在达成"最终协议"之前，对伊朗的封锁仍将维持，只补了一句"事情应该会进展得很快"。伊朗一侧则坚持，停止以色列在黎巴嫩的攻击是任何持久和平的前提，并把这轮战火重燃的责任归到美国头上，称以色列的举动"无法与美国的政策切割开来"。

俄乌方向则是暗线与明线交织。暗线上，乌克兰挫败了俄罗斯针对其军事情报部门发言人尤索夫的一桩暗杀图谋。明线上，联合国安理会于六月八日就俄罗斯近期的大规模袭击召开会议，联合国官员警告说，平民伤亡与破坏程度已达到这场战争近几个月来的最高位之一。与此同时，外交仍在推进，泽连斯基称他与美国特使威特科夫、库什纳在同日通了一次"非常正面"的电话。

**经济与市场**

美股这一侧暂无新的收盘可记，九日美国市场尚未开盘。需要补正的是周一的最终行情：上周五那场芯片股领跌的重挫之后，周一的反弹其实比盘中看上去要克制。标准普尔五百指数收涨百分之零点三，至七千四百零五点七三；纳斯达克综合指数收涨百分之零点八六，至两万五千九百二十九点六六；而道琼斯工业平均指数实际收跌八十点七七，即百分之零点一六，落在五万零七百八十六点零一。换句话说，芯片虽然回了血，但大盘的宽度偏弱，反弹并不齐整。市场下一个关口仍是六月十六至十七日的美联储议息会议。

**科技与 AI**

人工智能的算力叙事这两天落在硬件一端。英伟达首席执行官在一次演讲中称，公司面向数据中心的新款 Vera 中央处理器已进入全面量产，并把它定位成"新的主要增长引擎"，背后的推力是智能体应用的爆发，首批客户包括 Anthropic、OpenAI 与 SpaceX 旗下的人工智能部门。另一头，OpenAI 在六月八日围绕 ChatGPT 推出了一轮记忆能力的升级，意在让模型更连贯地记住与用户的过往交互，该能力正从付费版逐步铺向免费用户。归根结底，过去一两天没有重磅的前沿模型发布，这一阵的看点更多落在"把模型更深地嵌进产品与算力底座"这件事上。

**科学与气候**

菲律宾南部那场强震的伤亡仍在攀升。六月八日清晨发生在棉兰老岛萨兰加尼省外海的七点八级地震，死亡人数已从最初通报的数字升至至少三十五人，另有一百三十余人受伤、十余人失踪。在萨兰加尼省的格兰镇，一处山体滑坡冲垮民居，造成十三名村民遇难。这场地震恰好撞上新学年的第一天，超过三百二十万名学生受到影响，六千二百多所公立与私立学校被迫停课。
