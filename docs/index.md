# 2026-06-01 早间 brief

**地缘政治与国际关系**

香格里拉对话于五月三十一日在新加坡闭幕。在最后一场专题讨论中,菲律宾防长吉尔伯托·特奥多罗把菲律宾形容为日益碎片化的印太格局中的一个交汇点,并批评中国拒绝承认二〇一六年的南海仲裁裁决,同时就针对菲律宾等国的间谍与影响力活动表达关切。这场年度安全对话历来是各国掂量地区走向的重要场合,今年的焦点仍落在中美博弈与南海。

同在五月三十一日,美军南方司令部宣布在东太平洋对一艘船只实施打击,造成三名它所称的"毒品恐怖分子"死亡。近来美方直接动用军事手段打击海上贩毒目标的频次明显上升,这一做法本身在法律与外交层面也引来争议。

**经济与市场**

油价在五月底显著走低。上周五即五月二十九日,西得克萨斯中质原油下跌约百分之一点一,跌破每桶八十八美元,创约六周新低,整个五月累计下跌约百分之十六点二;布伦特原油在收于四月中旬以来的最低点后,回升至每桶九十三美元附近。背后的推手是美国与伊朗就延长停火、放开霍尔木兹海峡航运达成了初步共识。不过整个周末双方仍在就协议草案的细节往来交涉,到本周一,也就是六月一日,进展仍不明朗。

风险情绪则推动股市走高。同在上周五,衡量全球股票的明晟全球指数上涨约百分之零点二,亚洲股市上涨约百分之一、刷新历史高位,投资者继续加码人工智能这一主线,把主要股指推上纪录高点。

**科技与 AI**

台北的 COMPUTEX 2026 将于六月二日至五日举行。作为展前的重头戏,英伟达首席执行官黄仁勋于六月一日,也就是今天,发表主题演讲。外界关注的是,他会如何安排下一阶段的人工智能算力路线,以及与之配套的硬件与供货节奏,这往往为之后数月的行业风向定调。

**科学与气候**

中国载人航天有新进展。神舟二十一号乘组已于五月二十九日返回地球,完成天宫空间站本轮在轨任务与乘组轮换。随着空间站进入常态化运营,这类往返正逐渐成为按部就班的例行操作。

另在近地天体监测方面,包括阿波罗型小行星二〇二六 KV、二〇二六 KB1 以及阿莫尔型小行星二〇二六 JN 在内的多颗近地小行星,将于六月一日先后与地球近距离掠过,均处在安全距离之内,属于常规监测的范畴。

**社会与文化**

北美冰球赛场决出一个总决赛席位。卡罗来纳飓风队在东部决赛中以五场淘汰蒙特利尔加拿大人队,晋级二〇二六年斯坦利杯总决赛,这是该队自二〇〇六年以来首次闯入总决赛。
